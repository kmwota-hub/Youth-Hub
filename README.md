# 🌍 Youth Hub

Uniting youths across churches through events, collaboration, and shared purpose.

---

## 🚀 Live Site
Visit the hub here: [Youth Hub Live](https://kmwota-hub.github.io/Youth-Hub)

---

## 📖 About
The Youth Hub is designed to:
- Connect churches and youth groups.
- Share upcoming events instantly.
- Provide a central place for collaboration and community growth.

---

## 🗂️ Features
- **Home**: Introduction and highlights.
- **About**: Purpose and mission of the hub.
- **Events**: Upcoming events submitted by churches.
- **Churches**: List of active churches that have joined.
- **Admin**: Registration and event upload (Google/Email login).

---

## ⚙️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Firebase Authentication + Firestore
- **Hosting**: GitHub Pages
